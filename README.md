# 🛡️ Yunzai Blacklist Plugin

黑名单/授权管理插件，支持入群自动踢黑、自检踢黑、黑名单与管理名单查询以及主人授权，并在关键操作时私聊通知主人。

## 💡 安装教程 
### 使用 Git 安装（推荐）
1. 在Yunzai目录打开终端，运行
    ```sh
    git clone https://github.com/Sakura1618/yunzai-blacklist-plugin.git ./plugins/yunzai-blacklist-plugin
    ```
2. 重启 Yunzai-Bot 后即可使用。

### 手动安装
1. 下载 [yunzai-blacklist-plugin](https://github.com/Sakura1618/yunzai-blacklist-plugin) 的代码。
2. 将 yunzai-blacklist-plugin 放置在 Yunzai-Bot 的 plugins 目录下。
3. 重启 Yunzai-Bot 后即可使用。

## ✨ 功能
- 📥 入群检测：新成员在黑名单内时自动踢出（需 Bot 管理员权限）。
- 🔍 查黑自检：`#查黑` 扫描当前群成员，发现黑名单成员自动踢出并返回结果，并通知主人。
- 🌐 全群查黑：`#全群查黑` 主人一键扫描所有群，自动踢出黑名单成员，通知主人。
- ➕➖ 黑名单管理：`#加黑`、`#删黑` 添加/删除黑名单，操作后通知主人。
- 🧑‍💼 授权管理：主人可 `#黑名授权`/`#黑名取消授权` 指定可使用管理指令的 QQ（操作后通知主人）。
- 📃 列表查询：`#黑名单列表`、`#黑名单管理列表` 查看当前名单。
- 🆘 帮助：`#黑名单帮助` 查看完整指令列表。

## 📜 指令一览
- `#查黑`：扫描本群成员并自动踢出黑名单成员（需授权，通知主人）
- `#全群查黑`：扫描所有群并自动踢出黑名单成员（主人，通知主人）
- `#黑名单列表`：查看黑名单（需授权）
- `#黑名单管理列表`：查看管理名单（主人）
- `#加黑 <QQ>`：添加黑名单（需授权，通知主人）
- `#删黑 <QQ>`：删除黑名单（需授权，通知主人）
- `#黑名授权 <QQ>`：授权 QQ 使用黑名单管理指令（主人，通知主人）
- `#黑名取消授权 <QQ>`：取消授权（主人，通知主人）
- `#黑名单帮助`：查看帮助

## 🔐 权限说明
- 主人：始终有权限，并可授权/取消授权其他 QQ。
- 管理员名单：存储于 `data/blacklist_permission.json`，名单内 QQ 可使用黑名单管理和查询指令。
- 黑名单：存储于 `data/blacklist.json`。

## 🗂️ 目录结构
插件目录：
- `apps/common.js`：文件读写、授权工具函数
- `apps/notice.js`：入群检测踢黑
- `apps/check.js`：`#查黑` 群内自检踢黑
- `apps/manage.js`：黑名单增删、授权管理
- `apps/list.js`：名单查询
- `apps/help.js`：帮助指令

Yunzai 根目录(数据存储)：
- `data/blacklist.json`：黑名单
- `data/blacklist_permission.json`：管理名单

## 🚀 使用提示
1. 确保 Bot 在群内拥有管理员权限，以便踢黑成功。
2. 先由主人使用 `#黑名授权 <QQ>` 将管理者加入授权名单，再由其执行黑名单相关指令。
3. 修改黑名单/管理名单后可直接生效，无需重启；如遇缓存问题，可重启 Yunzai。

## ⚠️ 注意
- 勿将主人/关键账号加入黑名单，否则会被自动踢出。
- 踢人失败时会在群内提示并记录日志，需人工处理。
- 全群查黑功能目前不可用（适配器未提供/权限问题/风控等等），如果其他开发者有能力实现，欢迎提交 PR。
